```

    ↓ 国产替代

### 推理芯片（跑大模型）

```
英伟达 L40S/B200 → 仍是主流
    ↓ 国产替代
寒武纪(688256)  思元590 → 国产推理最强，已量产

### GPU 内部需要什么？

```
一块 GPU 芯片 = 计算单元 + HBM显存 + 互联接口

计算单元（需要5nm/3nm制程）
  └─ EUV光刻机(ASML，中国0台) ← 🔥🔥🔥🔥

  └─ 澜起科技(688008) PCIe 5.0 Retimer 全球第二
```
